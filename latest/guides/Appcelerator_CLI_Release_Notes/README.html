Appcelerator CLI Release Notes

Appcelerator CLI 4.1.0 Release Candidate

Appcelerator CLI 4.1.0 is a feature release, introducing new features, improvements and bug fixes. This release of the Appcelerator CLI includes version 4.1.0 of the NPM package and version 4.1.0 of the Core package.

Fixed Issues and Improvements

  • CLI-577: Need to add proxy logic to underlying appc process such as acs

  • CLI-649: Android build tool warning directs user to run "ti config"

  • CLI-676: CLI needs better support to handle proxy servers

  • CLI-678: Any network errors are treated as being offline using 'appc use'

  • CLI-699: Add banner for trial accounts

Known Issues

ETIMEDOUT is returned when you publish an Arrow app

Occasionally, when publishing an Arrow application, the CLI may return an ETIMEOUT error (CLI-654).

To workaround this issue, try republishing the application again.

Fail to Encrypt JavaScript Files on Windows

When building an application on Windows, the build fails at the encryption stage (CLI-658).

To workaround this issue, install the 32-bit version of Java.
