Arrow Builder Release Notes

Arrow Builder 1.0.1 - 29 May 2015

Arrow Builder 1.0.1 is a patch release, addressing high-priority issues from previous releases.

Fixed Issues

  • API-812: When using SSL via api doc, browser security exception is received

  • API-813: Using Count which matches 0 records should not return 404, but 200 with result of 0

Arrow Builder 1.0.0 - 21 May 2015

Initial release.

Known Issues

ETIMEDOUT is returned when you publish an Arrow app

Occasionally, when publishing an Arrow application, the CLI may return an ETIMEOUT error (CLI-654). To workaround this issue, try republishing the application again.
